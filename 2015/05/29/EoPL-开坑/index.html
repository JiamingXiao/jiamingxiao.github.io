EoPL 开坑

發表於 2015-05-29   |   分類於 EoPL   |  

昨天晚上看了下 Essentials of Programming Languages，感觉还真是有意思。虽然一开始的内容就是已经看烂了的 BNF 什么的，不过还是挺有意思的。这本书貌似就是教你写各种各样的 interpreter，而且貌似是用 scheme 写的。然后最近我也一直想好好学学 Erlang，于是我就想，干脆到时候用 Scheme 写一遍，然后再用 Erlang 重写一遍。

等会在 github 里开一个 repository，慢慢更新 EoPL 里的代码，Erlang 代码也应该会同步更新吧。嗯，坑先开着，到时候慢慢填……就酱吧。

然后代码都会放在这里。Have fun。
